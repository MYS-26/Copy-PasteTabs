Copy & Paste Tabs Extension

Options Page

Copy Behavior




Select tabs by holding Ctrl then click on the tabs you want to select.

Copy Output


Prevent website addresses with special characters (like é or ©) from being copied as unreadable code.
(e.g., copying "example.com/café" as "example.com/caf%C3%A9").

Copy tabs URLs as clickable links instead of plain text when using HTML or Custom copy formats. This is useful for pasting into programs like Word or email.

Paste Mode


Extracts and opens URLs from text. Supports standard web links (https, http, ftp, mailto, file, ssh) and non-standard URLs like localhost and Chrome-specific addresses. URLs without a protocol (e.g., example.com, www.example.com) won't be detected.

Treats each line as a URL and opens it without validating whether it's a URL.

Copy Format







Custom format
Customize your format by entering your desired template in the textarea below.
  • Use the variables $title and $url to dynamically include the title and URL in your template.
  • Add <br/> tags to create line breaks wherever needed.
Example: $title<br/>$url<br/>
